### React-приложение для отображения таблицы с данными

**Функционал**

- Редактирование информации
- Удаление элемента из таблицы
- Клиентская пагинация
- Данные в таблицу загружаются с сервера

**Использованные технологии**

- -create-react-app
- React Hooks
- Redux
- CSS Modules
- react-pagination
